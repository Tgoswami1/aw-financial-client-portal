# AW Financial Client Portal

## Overview

This project was developed as part of the Sagan AI Application Engineer assessment.

The goal was to design and build a solution that helps financial advisors automate the creation of quarterly client reports while reducing manual work and improving consistency.

---

## Problem Statement

The current workflow requires advisors to:

* Collect client financial data manually.
* Update Word and Canva templates.
* Perform financial calculations manually.
* Generate quarterly reports.
* Track historical reports.

This process is time-consuming and prone to human error.

---

## Solution

The AW Financial Client Portal provides a centralized platform for:

* Client management.
* Quarterly financial reporting.
* Automated calculations.
* Historical report tracking.
* PDF generation.

---

## Features

### Dashboard

* Client overview.
* Report statistics.
* Latest net worth metrics.

### Client Management

* Add new clients.
* Store financial information.
* Maintain client records.

### Quarterly Reports

* Select client.
* Enter financial information.
* Calculate savings targets.
* Calculate net worth.

### PDF Export

* Generate downloadable financial reports.

### Report History

* View previously generated reports.
